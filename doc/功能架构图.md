## Web 版 (Spring Boot) 对应关系

| 层次 | Swing 版 | Web 版 |
|------|----------|--------|
| 表现层 | `scut.ui.*` (JFrame/JPanel) | `scut.controller.*` + `templates/*.html` |
| 业务层 | `scut.service.*` | 完全复用 |
| 数据层 | `scut.dao.*` + `scut.util.*` | 完全复用 |
| 实体层 | `scut.entity.Student` | 完全复用 |
